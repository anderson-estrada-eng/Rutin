# Tarea Vínculo ¿Horario? Duración Inicio Fin Notas

Vínculo: clic para elegir hoja. 📁 carpetas · arrastra pestañas a una carpeta. Clic en carpeta para desplegar.

Cómo funciona

  • ¿Horario? Si está marcado, la tarea entra en la línea del tiempo desde el inicio del día.
  • Duración: escribe 30, 45m, 1h, 1h30 o 1:30.
  • Reordenar: arrastra la fila; la duración se mueve y las horas de abajo se ajustan solas.
  • Límites: empieza a las 6:00 AM; no debería pasar de las 4:00 PM. La siguiente madrugada referencia es 7:00 AM.
  • Pestañas: clic para abrir; ✎ o doble clic para renombrar; + añade hoja vacía; × elimina.
  • Vínculo: clic en la celda → elige una hoja (sin escribir sheet:) o pega una URL externa.
  • Carpetas: botón 📁 para agrupar hojas; arrastra una pestaña sobre una carpeta; clic en la carpeta para desplegar.
  • Texto largo: si la celda corta el texto, al pasar el mouse ves el completo; clic en el globo o en la celda para editarlo ahí.
  • Acceso: la app pide usuario y contraseña; sin sesión no se puede ver ni la API.
